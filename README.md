# 📊 Unemployment Data Analysis in India (COVID-19 Impact)

-> **NOTE**: This repository contains the Task 2 project completed during my Data Science Internship at @CodeAlpha (https://www.codealpha.tech)

## 🎯 Objective
This project analyzes India's unemployment rate, focusing on the economic shock caused by the COVID-19 pandemic. The data pipeline involves cleaning, exploratory data analysis (EDA), and interactive visualization using Python.

## 🛠️ Tech Stack
* **Language:** Python
* **Libraries:** Pandas, NumPy, Matplotlib, Seaborn
* **Environment:** Jupyter Notebook / VS Code

## 💡 Key Insights & Policy Recommendations
1. **The COVID-19 Shock:** The data reveals a massive, unprecedented spike in unemployment between April and June 2020, aligning perfectly with the strict nationwide lockdown. Graph is located in the graph folder, path (graph/Unemployment%20Rate%20in%20Timeline%20The%20COVID-19%20Impact.png)
2. **Urban vs. Rural Divide:** Urban areas experienced higher average unemployment rates and greater volatility compared to rural sectors. 
   * *Policy Insight:* Future crisis management requires specialized "Urban Employment Guarantee Schemes," as urban casual laborers lack the safety nets present in rural agricultural sectors. Graph is located in the graph folder, path (graph/Unemployment%20Distribuition%20Urban%20vs%20Rural%20Sectors.png)
3. **Regional Disparities:** States like Tripura, Haryana, and Jharkhand showed consistently higher localized unemployment during the crisis.
   * *Policy Insight:* Economic stimulus packages should not be universal; they must be targeted geographically toward these high-risk states to stabilize regional economies. Graph is located in the graph folder, path (graph/Top%2010%20Most%20Affected%20States%20During%202020.png)
